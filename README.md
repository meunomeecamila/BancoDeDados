# Banco de Dados

Este repositório reúne todo o material produzido durante a disciplina de **Banco de Dados**, incluindo resumos, exercícios, atividades avaliativas, trabalhos práticos e provas. O objetivo é servir como um material de apoio para estudos, revisões e consultas futuras.

## Estrutura do Repositório

### 📚 Resumos
Contém os resumos elaborados durante as aulas, organizados de acordo com a sequência dos slides da disciplina. Os arquivos abordam desde os conceitos fundamentais de bancos de dados até temas mais avançados, como indexação, otimização de consultas e controle de concorrência.

### 📝 Exercícios
Reúne listas de exercícios resolvidos e propostos ao longo da disciplina, abordando conteúdos como modelagem de dados, álgebra relacional, SQL, indexação, transações e otimização de consultas.

### 💻 HO (Hands On)
Contém as atividades práticas desenvolvidas durante as aulas (Hands On). Essas atividades têm como objetivo aplicar os conceitos teóricos em situações práticas utilizando SQL e ferramentas de modelagem de bancos de dados.

### 📄 AS (Atividades Avaliativas)
Pasta destinada às atividades avaliativas realizadas durante o semestre. Cada atividade aborda diferentes conteúdos estudados na disciplina e representa parte da avaliação contínua.

### 📑 Provas
Contém as provas aplicadas durante a disciplina, juntamente com resoluções, correções e materiais utilizados para estudo.

### 🏗️ TP (Trabalho Prático)
Reúne todos os arquivos relacionados ao Trabalho Prático da disciplina, incluindo levantamento de requisitos, modelagem conceitual, modelo lógico, implementação do banco de dados, scripts SQL e documentação do projeto.

### 🎓 Aulões
Material utilizado para revisões gerais da disciplina, contendo resumos, exercícios e questões comentadas para preparação para provas.

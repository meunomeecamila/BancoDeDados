# 📚 Resumos de Banco de Dados

Esta pasta reúne os resumos que produzi durante a disciplina de **Banco de Dados**. Todo o material foi elaborado com base nas aulas ministradas pelo professor e organizado de acordo com a sequência dos slides utilizados em sala.

Os slides originais da disciplina podem ser encontrados no repositório do professor:

🔗 https://github.com/wladbrandao/DBSystems.git

---

| Arquivo | Conteúdo | Slides |

## Objetivo

O objetivo deste material é servir como um guia de estudos para revisões da disciplina de Banco de Dados, reunindo os principais conceitos apresentados em aula de forma resumida e organizada.

Os resumos não substituem os slides oficiais nem a bibliografia da disciplina, mas funcionam como um material de apoio para consultas rápidas e preparação para provas.
